# Vigilant Owl Backend

Vigilant Owl is a service aimed at providing parents with non-intrusive insights into their children's group conversations across various social media platforms. This repository contains the backend implementation using TypeScript, Node.js, and Express.

## Features

- User management (create, retrieve users)
- Basic RESTful API structure
- Type safety with TypeScript
- Easily extendable architecture

## Technologies Used

- **Node.js**: JavaScript runtime for building fast and scalable applications.
- **Express**: Web framework for Node.js to build APIs.
- **TypeScript**: Superset of JavaScript that adds static types.
- **Supabase**: Backend-as-a-Service for database management and authentication (if used).
- **whatsapp-web.js**: Library for interacting with WhatsApp Web.
